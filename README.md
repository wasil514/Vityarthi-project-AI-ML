Vityarthi Project – Healthy Programmer Reminder System

Overview  
This project is a simple CLI-based Healthy Programmer Reminder System built in Python.  
It helps the user maintain healthy habits while working on a computer by giving reminders to drink water, relax eyes, and perform physical exercise at regular time intervals.  
The program also stores activity records in log files with timestamps.

Features  

Reminder mode:  
Play sound reminder to drink water  
Play sound reminder for eye exercise  
Play sound reminder for physical exercise  

Logging system:  
Store water drinking records  
Store eye exercise records  
Store physical exercise records  

User interaction:  
User confirms task completion using input  
Each entry is timestamped for tracking purposes  
Separate log files for each activity  
(e.g., Water_records.txt, Eyes_log.txt, Exercise_records.txt)

Technology Stack  

Python 3.x  
Pygame module for sound  
Built-in datetime module  
Time module for interval control  
File I/O for data storage  
Command-Line Interface (CLI)

How to Run  

Clone the repository:  
git clone https://github.com/wasil514/Vityarthi-project-AI-ML.git  

Go to project folder and run the program using Python.
